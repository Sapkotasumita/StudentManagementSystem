# Student Management System (JavaFX)
A *Student Management System* developed using *Java*, *JavaFX*, and *Scene Builder*, with data stored locally using *CSV files*.  
This project provides an interactive desktop-based solution for managing student-related activities efficiently.

# Overview
The Student Management System is designed to simplify academic data handling by providing modules such as student records, attendance tracking, library management, counseling, tasks, and reports.  
The system uses a *JavaFX GUI* and stores data in *CSV files*, making it lightweight and easy to maintain without a database.

# Features
-  User Authentication (Login & Signup)
-  Student Management (Add, Update, Delete, Search)
-  Attendance Management
-  Library Management
-  Counseling Records
-  Question Bank Management
-  Reports Generation
-  CSV-based Data Storage
- User-friendly GUI built with JavaFX & Scene Builder

# Technologies Used
- *Java (JDK 11+)*
- *JavaFX*
- *Scene Builder*
- *CSV Files* for data storage
- *FXML & CSS* for UI design

# Requirements
- Java JDK 11 or higher
- JavaFX SDK
- Scene Builder
- IDE (IntelliJ IDEA / Eclipse / NetBeans recommended)
